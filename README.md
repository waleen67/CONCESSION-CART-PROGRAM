# 🛒 Simple Python Food Ordering System

This is a simple Python program that simulates a food ordering system.

## 📋 Features
- Display a menu with prices
- Allow user to select items
- Store selected items in a cart
- Calculate total price
- Show final order summary

## 🧑‍💻 How it works
1. The program displays a menu
2. The user inputs food names
3. Items are added to the cart
4. Typing `q` exits the program
5. The total price is calculated and displayed
